# 数据分析平台

一个用于采集和分析数据的 Web 平台，支持多用户管理、数据采集、数据分析和可视化展示。

## 功能特点

- 多用户系统，支持管理员和普通用户角色
- 自动数据采集，支持定时更新
- 数据可视化展示，包括趋势图、饼图等
- UID 管理，支持备注名称
- 灵活的价格设置系统
- 响应式设计，支持移动端访问

## 系统要求

- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Web 服务器（Apache/Nginx）
- 启用 PDO 和 PDO_MySQL 扩展
- 启用 cURL 扩展

## 安装步骤

1. 将代码部署到 Web 服务器目录
2. 确保 `config`、`storage` 目录可写
3. 访问网站，系统将自动跳转到初始化页面
4. 按照提示配置数据库信息
5. 创建管理员账号
6. 完成安装
